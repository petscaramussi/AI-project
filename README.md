# AI-project
 
### Order your pizza through an interactive chat

# Getting started

1. Go to project folder (client) and install dependencies:
 ```bash
 npm install
 ```
 
2. Launch development server, and open `localhost:4200` in your browser:
 ```bash
 npm start
 ```
